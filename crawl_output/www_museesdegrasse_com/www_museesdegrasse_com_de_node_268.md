## Qu'est-ce-que l'accessibilité numérique ?
L'accessibilité numérique correspond au fait que des personnes handicapées peuvent accéder à un service numérique grâce aux dispositifs techniques qui leur sont propres (terminaux braille, dispositifs de synthèse vocale, système de commande remplaçant le clavier ou la souris, ...).
Source : 
Pour en savoir plus : www.braillenet.org : association pour le développement de l'accessibilité des contenus sur le web.
## Les raccourcis claviers ou "accesskeys"
Les raccourcis claviers suivant vous permettront d'aller plus rapidement à certaines parties de la page ou du site :
Règles d'accessibilité : accesskey="0" Accueil : accesskey="1" Plan du site : accesskey="3" Accès direct au moteur de recherche interne : accesskey="4" Nous envoyer un mail : accesskey="7" Mentions légales : accesskey="8"
## Mode d'emploi
Les combinaisons de touches pour valider ces raccourcis diffèrent selon les navigateurs, c'est pourquoi nous listerons ci dessous les procédures à suivre pour activer ces raccourcis claviers dans les principaux navigateurs :
 * Mozilla : Appuyez simultanément sur la touche Alt, sur la touche maj (pour les chiffres) et sur une des touches du clavier (pas le pavé numérique). Vous vous rendrez directement à l'endroit voulu.
 * IE6 : Appuyez simultanément sur la touche Alt, sur la touche maj (pour les chiffres) et sur une des touches du clavier (pas le pavé numérique) et ensuite appuyez sur la touche "Entrée" pour vous rendre à l'endroit voulu.
 * Opera supporte les accesskeys depuis sa version 7, via la combinaison de touches Esc + Shift et [accesskey], sur tous les systèmes d'exploitation.

## Mises en forme alternatives
La mise en forme du site est séparée de son contenu grâce à l'utilisation des feuilles de style. Cela nous permet de proposer des mises en forme différentes. Depuis le menu Affichage des navigateurs supportant les styles alternatifs : menu "Utiliser une feuille de style" dans Netscape 7, menu "Style" dans Opera 7... Dans les versions actuelles de ces navigateurs, le choix d'une présentation autre que celle par défaut n'est cependant pas persistant : il doit être répété pour chaque page consultée.
## Autres aides à la navigation
Des liens d'évitements sont placés en tête de la page. Ils permettent de se rendre directement à certaines parties de la page. La têtière permet par un simple clic de retourner sur la page d'accueil. Le fil d'ariane permet de remonter dans l'arborescence. On peut naviguer d'un lien à l'aide de la touche de tabulation. L'ordre des 5 premiers liens est le suivant :
 * lien vers la page donnant la politique d'accessibilité (cette page)
 * lien permettant d'aller directement au menu
 * lien permettant d'aller au contenu de la page
 * lien permettant d'aller au formulaire de contact
 * lien permettant de retourner à l'accueil.

## Respect des standards
Le code XHTML est conforme à la Spécification XHTML 1.0. Il est testé à l'aide du validateur HTML du W3C. Le code CSS est conforme à la Spécification CSS2. Il est testé à l'aide du validateur CSS du W3C. Un bug connu du validateur (lié à l'encodage des caractères) empêche cependant la validation CSS directe depuis les fichiers HTML. Les URL à soumettre au validateur CSS sont donc celles des feuilles de style elles-mêmes.
Si malgré nos efforts, des erreurs étaient détectées suite à des validations, nous vous remercions de nous les signaler.
